# PASS-IT-ON

A simple donation landing page project.

## Project Structure

- `Pass-it-on/pass-it-on.html` — main HTML page for the donation site
- `Pass-it-on/style.css` — page styling and layout
- `Pass-it-on/script.js` — interactive behavior and client-side logic
- `Pass-it-on/pass-it-on-logo.jpeg` — logo image used by the page

## Description

This project contains a single static web page designed to share information about donations and encourage users to contribute. It includes the page markup, styling, and JavaScript needed to make the site interactive.

## How to view

1. Open `Pass-it-on/pass-it-on.html` in a web browser.
2. Or host the `Pass-it-on/` folder using a static server to preview the page locally.
